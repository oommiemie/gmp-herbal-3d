# 🏭 GMP Herbal Plant 3D — ผังจำลอง 3 มิติ โรงงานผลิตยาสมุนไพร

เว็บแอปจำลองอาคารผลิตยาสมุนไพรตามหลัก GMP ขนาด **28 × 12 ม.** แบบ 3 มิติ
ไฟล์เดียวจบ (`index.html`) ไม่ต้องมีเซิร์ฟเวอร์/ติดตั้งอะไรเพิ่ม — เปิดในเบราว์เซอร์ได้ทันที

## ฟีเจอร์

- **4 มุมมอง** — มุมบน (แปลน + เส้นบอกระยะ), ไอโซเมตริก, หมุนรอบ 360°, เดินสำรวจระดับสายตา (WASD + ระบบชนผนัง, แผนที่ย่อ, จอยสติ๊กบนมือถือ)
- **ทัวร์นำชมอัตโนมัติ 14 จุด** ไล่ตามลำดับกระบวนการ พร้อมคำบรรยาย GMP
- **เส้นทางการไหล (GMP Flow)** 4 สาย: วัตถุดิบเข้าผ่านแอร์ล็อกสองชั้น · สายยาแห้ง (ผสม→แคปซูล→ขัดเม็ด→ใส่ขวด→ติดฉลาก→คลัง) · สายยาน้ำ (ผ่านแอร์ล็อกเฉพาะ) · เส้นทางบุคลากร — แสดงแบบ x-ray ทะลุผนังในมุมแปลน
- **ประตูบานสวิง** เปิดอัตโนมัติเมื่อเดินเข้าใกล้ พร้อมช่องกระจกและแผ่นกันกระแทก
- **พื้นอิพ็อกซีทั้งอาคาร** + พื้นสโลปเทลาดเข้าตะแกรงระบายน้ำตามมุมห้องจริง (ห้องเปียก 5 ห้อง)
- **ห้องระบบน้ำ RO** แยกนอกไลน์ผลิต ประตูเข้าจากภายนอก
- เครื่องมือ: **วัดระยะ**, คลิกห้องดูข้อมูล/ขนาด/พื้นที่, **ปรับระดับกราฟิกอัตโนมัติตาม FPS** (สลับเองได้: สูง/กลาง/ต่ำ)
- เรนเดอร์เกรดพรีเซนต์: SAO ambient occlusion + FXAA (ระดับสูง), environment reflections, ประตูบานสวิงอนิเมชัน

```

src/app2.js     ← เฟอร์นิเจอร์/เครื่องจักร ภายนอก เพดาน

```
